\chapter*{Abstract}
This project presents the design and development of \textbf{Web Core View}, a modern web platform for the management of security video recorders. The project arises from a twofold need at Lanaccess Telecom, SA: on one hand, to replace its legacy web management platform, originally built in the late 1990s using obsolete technologies such as \textbf{ActiveX} controls, with a modern solution that ensures ubiquity, security and sustainability; on the other hand, to provide a standalone, lightweight version aimed at small businesses that manage a single video recorder, without requiring a centralized application connected to the entire device fleet. The solution has been implemented as an embedded web administration interface based on a \textbf{coordinated microservices} architecture, using \textbf{Angular 19} and the new reactivity paradigm based on \textbf{Signals}. The two most significant technological pillars of the project are the dynamic configuration engine (\textit{Schema-driven}), which allows decoupling the interface from firmware evolution, and the integration of the \textbf{WebRTC} protocol for low-latency video streaming ($<300$ms) without the need for external plugins. The results validate that the web solution matches or improves the performance of the company's legacy tools.
