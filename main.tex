\chapter*{Resum}
Aquest projecte presenta el disseny i desenvolupament de \textbf{Web Core View}, una plataforma web moderna destinada a la gestió de videogravadors de seguretat. El treball neix d'una doble necessitat de Lanaccess Telecom, SA: d'una banda, substituir la seva plataforma web de gestió \textit{legacy}, construïda originalment als anys 90 amb tecnologies avui obsoletes com els controls \textbf{ActiveX}, per una solució moderna que garanteixi ubiqüitat, seguretat i sostenibilitat; de l'altra, disposar d'una versió autònoma i simplificada orientada a empreses petites que gestionen un únic videogravador, sense necessitat de dependre d'una aplicació centralitzada connectada a tot el parc de dispositius. La solució s'ha implementat com una interfície web d'administració embedded basada en una arquitectura de \textbf{microserveis coordinats}, utilitzant \textbf{Angular 19} i el nou paradigma de reactivitat basat en \textbf{Signals}. Els dos components tecnològics més rellevants del projecte són el motor de configuració dinàmic (\textit{Schema-driven}), que permet desacoblar la interfície de l'evolució del \textit{firmware}, i la integració del protocol \textbf{WebRTC} per a la visualització de vídeo de baixa latència ($<300$ ms) sense necessitat de \textit{plugins} externs. Els resultats validen que la solució web iguala o millora el rendiment de les eines clàssiques.

\chapter*{Abstract}
This project presents the design and development of \textbf{Web Core View}, a modern web platform for the management of security video recorders. The project arises from a twofold need at Lanaccess Telecom, SA: on one hand, to replace its legacy web management platform, originally built in the late 1990s using obsolete technologies such as \textbf{ActiveX} controls, with a modern solution that ensures ubiquity, security and sustainability; on the other hand, to provide a standalone, lightweight version aimed at small businesses that manage a single video recorder, without requiring a centralized application connected to the entire device fleet. The solution has been implemented as an embedded web administration interface based on a \textbf{coordinated microservices} architecture, using \textbf{Angular 19} and the new reactivity paradigm based on \textbf{Signals}. The two most significant technological pillars of the project are the dynamic configuration engine (\textit{Schema-driven}), which allows decoupling the interface from firmware evolution, and the integration of the \textbf{WebRTC} protocol for low-latency video streaming ($<300$ms) without the need for external plugins. The results validate that the web solution matches or improves the performance of the company's legacy tools.
